공공데이터포털 API 키

data.go.kr에서 발급받은 일반 인증키(Decoding)를 입력하세요.

키가 없다면 data.go.kr에서 "경기도_버스도착정보 조회" 검색 → 활용신청 → 마이페이지에서 발급받은 디코딩 키를 복사해 붙여넣으세요.

먼저 API 키를 등록해주세요

경기버스 실시간 정보를 받아오려면 공공데이터포털 인증키가 필요해요.

data.go.kr → "경기도_버스도착정보 조회" 검색 → 활용신청 → 일반 인증키(Decoding) 복사
